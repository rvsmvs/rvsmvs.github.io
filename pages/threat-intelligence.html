Threat Intelligence Overview

From indicators to behaviors, map and validate findings for actionability.

Indicators of Compromise (IoCs)

Track file hashes, domains, IPs, URLs, and registry keys. Consider quality (false positive risk) and lifetime (volatility) when using IoCs operationally.

  • File hashes: SHA-256 preferred; note packer influence and versioning.
  • Network: domains, IPs, URLs; monitor DNS TTLs and hosting churn.
  • Host artifacts: registry keys, services, scheduled tasks, mutexes.
